# ProfEx

ProfEx is an AI-powered research assistant that helps users discover, analyze, and understand academic papers from arXiv.

## Features

* Search and retrieve research papers from arXiv
* Ask questions about research papers in natural language
* Generate concise summaries of complex papers
* Extract key findings, methodologies, and conclusions
* Compare multiple research papers
* Fast and intelligent paper exploration using AI

## Use Cases

* Literature review
* Research exploration
* Learning new topics
* Understanding complex academic papers
* Finding relevant research quickly

## How It Works

1. User enters a research query or question.
2. ProfEx searches relevant papers from arXiv.
3. The AI processes and analyzes the papers.
4. Results are returned as summaries, insights, and answers.

## Installation

```bash
git clone https://github.com/akash-kumarrr/ProfEx.git

cd profex

pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

Then ask questions such as:

* "Latest papers on Agentic AI"
* "Summarize recent research on RAG systems"
* "Compare top papers on multimodal LLMs"
